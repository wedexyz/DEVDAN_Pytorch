# DEVDAN_Pytorch
Pytorch implementation of Deep Evolving Denoising Autoencoder (DEVDAN) published in Neurocomputing, 2020 titled "DEVDAN: Deep evolving denoising autoencoder. Neurocomputing 390: 297-314 (2020)"

datasets can be downloaded from "https://drive.google.com/drive/folders/1Kdkcdagctx1tX3bGIPo85XWIYKLAoMww?usp=sharing"

To run the code, one can go directly to .ipynb files. For example, if one wants to try DEVDAN in SEA problem, one can go to DEVDAN_sea.ipynb.

Our numerical results can be seen in the .html files.
